# Demo comercial lista para Vercel

## Ejecutar en local
```bash
npm install
npm run dev
```

## Publicar en Vercel
1. Sube esta carpeta a un repositorio de GitHub.
2. Entra en Vercel.
3. Pulsa **Add New Project**.
4. Importa el repositorio.
5. Pulsa **Deploy**.

Vercel detectará Vite automáticamente y publicará la demo.

## Build settings esperados
- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
